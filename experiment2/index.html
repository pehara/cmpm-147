Experiment 2 - Living Impressions

Description

In this experiment, I tried to generate a realistic snowy scene. I tried to mimic the dynamic characteristics of nature in real life. I wanted the snow to be moving, the trees to slowly drift and for it feel sort of dream like.

Technical

This project uses the p5.js package to build and control visual objects on an HTML canvas. It helps with the functions of rendering the sky gradient, mountains, forests, and snow layers. The sky gradient is made out of blended rectangles, mountains are rendered as randomly shaped polygons with the beginShape() and vertex() methods, trees are illustrated using triangles that are randomly placed throughout the canvas and vary in size and finally I use ellipses to replicate snowflakes.I also have user input by using mouse movement.

Reflection

I did all the work of the project by myself, I did have the aid of some online resources when looking up the library for creating my scene as well as certain effects. I think I did a great job at implementing the work into something I can confidently say looks good. I did want to have more diversity with my trees like random colors and shapes however I struggled with my implementation of that so I decided to just revert to a simple triangle. I would say my high, however, was my implementation of the gradient because it went fairly smoothly and had a really nice outcome in my opinion.
