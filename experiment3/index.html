Experiment 3 - Alternate Worlds

Description

The purpose of this experiment is to explore my version of alternate worlds within a digital environment.

Technical

his code is for a project where I had to create a grid-based world with different types of terrain and dynamic features. I used look-up tables to define the appearance of snow, ice, and chests on the grid. The generateGrid function is responsible for setting up the grid, adding rooms, hallways, and randomly placing chests. To make it more interactive, I implemented snow animations using the generateSnowAnimations function, which adds movement to the snow tiles. Then, the drawGrid function renders everything, from the background tiles to the animated snow and randomized terrain textures. I also included a function for shuffling arrays to ensure random tile placement. Overall, it was a fun assignment that allowed me to create a visually interesting and dynamic grid-based world.

Reflection

I did all the work of the project by myself, I did have the aid of some online resources when looking up the library for creating my scene as well as a algorithm to shuffle my tiles. I feel that my implementation was goof and I think that the final outcome came out really well. I did want to include more of a transition like a gradient for my water and the island however I struggled to implement that so I decided to keep it simple. My high for the project was letting my tiles shuffle through the array of coordinates I created creating really cool snowing effect.
