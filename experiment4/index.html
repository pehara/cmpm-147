Experiment 4 - Infinite Worlds

Description

This experiment aims to create an infinite snowy world with variations in terrain and interactive elements.

Technical

This experiment uses Perlin noise to generate terrain height values. Based on these values, different colors represent snowy plains, hills, and mountains. Icy lakes are randomly added to the terrain within a specific height range. Each tile reacts to clicks, changing its appearance.

Here's how it works:

  • A world seed is defined based on the input key using XXH hashing.
  • Each tile's height is determined using Perlin noise.
  • Different colors are assigned for snowy terrain based on height values.
  • Icy lakes are randomly added within a specific height range.
  • Clicking on tiles changes their appearance.
